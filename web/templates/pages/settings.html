Usage Instructions

Using cURL:

curl -H "session: YOUR_TOKEN_HERE" https://kindredcard.com/api/v1/contacts

Using Python:

import requests
headers = {'session': 'YOUR_TOKEN_HERE'}
response = requests.get('https://kindredcard.com/api/v1/contacts', headers=headers)

Using PowerShell:

$myURI = "https://kindredcard.com"
$myHeaders = @{"session" = "YOUR_TOKEN_HERE"}
Invoke-RestMethod -Uri "$myURI/api/v1/contacts" -Headers $myHeaders
